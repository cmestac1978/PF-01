# PF-01
Proyecto Phtyon-Fundamentals
Aplicación desarrollada en Streamlit que integra conceptos revisados en las sesiones relacionados con :
-Definición de variables
-Definicíón de titulos
-Creación de menus de opciones
-Invocación a cada opción de menú
-Condiciones
-Usar objetos en memoria
-Uso de arreglos
-Clasen/métodos
-Validaciones

El desarrollo comprende
App.py (programa)
Requirements.txt (lista de librerías usadas , si deseamos usar una versión especifica con pip list identificas la versión)
